# 🛒 Telegram Mini App — Уютное Кафе

Современный Telegram-бот с интегрированным веб-приложением (Mini App) для автоматизации заказов в кафе или ресторане. Проект готов к деплою и демонстрации заказчикам.

## ✨ Основной функционал
* **Интерактивная витрина:** Удобный веб-интерфейс (Mini App) прямо внутри Telegram для выбора блюд и напитков.
* **Категории меню:** Разделение товаров на категории (Бодрящий Кофе, Свежая Выпечка).
* **Динамическая корзина:** Автоматический подсчет стоимости заказа в реальном времени.
* **Связь с Telegram API:** Использование официального SDK для бесшовной передачи данных заказа обратно в чат с ботом.
* **Безопасность:** Архитектура проекта полностью скрывает конфиденциальные токены через переменные окружения.

## 🛠 Технологический стек
* **Backend:** Python 3.x, `aiogram` (или ваша библиотека) для логики Telegram-бота.
* **Frontend:** HTML5, CSS3, JavaScript (Telegram WebApp API) для витрины.
* **База данных:** SQLite (автоматически создается при первом запуске).

## 🚀 Инструкция по локальному запуску

### 1. Подготовка окружения
Склонируйте репозиторий и перейдите в папку проекта:
```bash
git clone https://github.com
cd cafe_bot
```

Создайте и активируйте виртуальное окружение, затем установите зависимости:
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate

pip install -r requirements.txt
```

### 2. Настройка переменных окружения
1. Переименуйте файл `.env.example` в `.env`.
2. Откройте файл `.env` и вставьте ваш токен бота, полученный от `@BotFather`:
```text
BOT_TOKEN=ваш_токен_бота_здесь
```

### 3. Настройка WebApp ссылки
В файле конфигурации (`config.py` или `handlers.py`) укажите вашу ссылку на развернутый фронтенд проекта (например, GitHub Pages или Vercel):
```python
YOUR_WEBAPP_URL = "https://ваш_логин.github.io/название_репозитория/frontend/"
```

### 4. Запуск бота
Выполните команду в терминале:
```bash
python main.py
```
При первом запуске скрипт автоматически создаст локальную базу данных `cafe.db`.

## 📁 Структура проекта
* `frontend/` — Файлы веб-интерфейса (витрина магазина).
* `config.py` — Конфигурация и загрузка настроек окружения.
* `database.py` — Логика работы с базой данных SQLite.
* `handlers.py` — Обработчики команд Telegram и событий WebApp.
* `main.py` — Точка входа для запуска бота.
* `.env.example` — Шаблон настроек для безопасного деплоя.
* `.gitignore` — Список исключений, защищающий токены и кэш от попадания в публичный доступ.
